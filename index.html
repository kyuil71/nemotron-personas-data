한국 전용 Nemotron-Personas 다단계 추출기

NVIDIA Nemotron-Personas-Korea 데이터셋에서 조건에 맞는 합성 페르소나를 수집합니다. 1차는 성별·나이·지역·학력 같은 정확 조건으로 1단계당 5,000명씩, 최대 25,000명까지 누적 수집하고, 2차는 프로그램 안에 보관된 전체 1차 후보군을 키워드로 다시 정밀 필터링합니다.

  • 1차동일 조건에서 1~5단계 누적 수집
  • 2차누적 후보 전체에서 키워드 정밀 필터
  • 저장엑셀 또는 프로젝트 JSON 저장

표시 라벨은 한국어로 보이지만, 수집 조건은 Hugging Face 원본 컬럼명과 원문값을 사용합니다.

1

1차 정확 조건

성별, 나이, 지역, 학력처럼 명확한 조건은 먼저 서버에서 줄입니다. 필요 없으면 상관 없음으로 두세요.

2

1차 후보 수집: 1~5단계 누적

동일한 정확 조건에서 검색 구간을 나누어 1단계당 5,000명씩 수집합니다. 5단계까지 완료하면 최대 25,000명의 1차 후보군이 프로그램 안에 누적됩니다.

1차 단계에서는 키워드 검색을 사용하지 않습니다. 각 단계는 같은 조건의 다른 구간을 가져오며, 수집된 모든 1차 후보를 대상으로 2차 필터링을 진행합니다.

토큰은 브라우저 메모리에만 임시로 두며 Hugging Face API 호출에만 사용됩니다.

3

2차 내부 필터링

엑셀 업로드 없이, 1~5단계에서 누적 수집한 1차 후보 전체 안에서 다시 검색합니다.

먼저 1차 후보를 수집하면 2차 필터를 사용할 수 있습니다.

쉼표·줄바꿈·슬래시로 여러 키워드를 입력하세요.

제외 키워드가 하나라도 포함되면 최종 결과에서 제외됩니다.
